### Helpful GitHub Issues.

Recently I was working on the paper *Nested Learning* by GDM. Since there was no official code for the paper, which was reasonable given that Google they use their inside infra, I turned to GitHub for community effort.

When I was going through some reproduction repositories, I found the issues people discussed were really insightful. They even foster my understanding of the original paper. Here I really appreciate the effort of [Kyle's repo](https://github.com/kmccleary3301/nested_learning) and [weipeilun's](https://github.com/weipeilun/Nested-Learning-Pytorch).

Just found that GitHub Issues is a good place to better understand top papers!
